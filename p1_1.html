자료형


소스모드에는 자료형이 많습니다 하지만 딱 5가지만 살펴볼것입니다 왜냐하면 이해하기 어려울수도 있기때문입니다
_________________________________________

변수 선언하는 방법
_________________________________________
1.자료형 이름;
2.자료형 이름 = 값;

1번째 방법은 초기화를 하지 않는 방법입니다
2번째 방법은 초기화를 한 방법입니다

초기화를 하지 않으면 소스모드에서 저절로 0 으로 초기화를 시켜줍니다
값을 넣을때는 자료형에 맞게 넣어야 합니다
값이 자료형 사이즈를 초과하면 오버 플로우가 생깁니다! 반대로 값이 자료형 사이즈를 미만하면 반대로 언더 플로우가 생깁니다!
이름은 꼭 영어 이여야 합니다 한국어, 러시아어, 폴란드어, 유럽어 등은 안됩니다
그리고 !, #, $, % 등 특수기호 혹은 이름앞에 숫자가 붙으면 에러가 납니다

에러 예제:

float abc = 5; //5 는 정수라서 5.0 으로 해야 맞습니다
int 1open = 005; //숫자가 앞에있으면 안됩니다
bool ?help = true; //특수문자가 있으면 안됩니다
char word = "ABCD"; //자료형 사이즈를 초과했습니다 그리고 "" 이 표시는 const char 형 입니다 

예제:

float very_good = 5.0;
bool i_feel_good = true;
bool NOPE = false;
int age = 32;
char word = 'a';

변수에 값 대입 하는 방법
_________________________________________


변수를 선언할때는 자료형 이름; 혹은 자료형 이름 = 값; 으로 하고 
값을 대입 할때는 이름 = 값; 으로 합니다 대입할때는 무조건 선언이 된 이름이여야합니다

예제:

int value;
value = 10;

float floatvalue;
floatvalue = 34.0;

bool Server = false;
Server = true;

char word;
word = 'A';

void 형
_________________________________________

말 그대로 입니다
아무것도 없다는 뜻입니다
void 형으로 변수 선언은 아에 못합니다
void 형 사이즈는 0 비트 입니다
_________________________________________


int 형
_________________________________________

정수형을 나타내는 기호 입니다
다른값이 들어가면 오류가 납니다. 예를들어 실수(實數)
정수형 사이즈는 4 바이트 입니다 최대 -2,147,483,648 에서 부터 2,147,483,647‬ 까지 저장 가능합니다

예제:

int VeryGood = 501;
int Nein = 591;
int TF2IsGood = 10000;
itn OverPower = 2147483647‬;
_________________________________________

float 형
_________________________________________

실수형을 나타내는 기호 입니다
다른값이 들어가면 오류가 납니다. 예를들어 정수
실수형 사이즈는 4 바이트 입니다 최대 1.175494e-38 에서 부터 3.402823466e+38‬ 까지 저장 가능합니다

e-(값)
e+(값)
는 10^-(값), 10^+(값) 과 같습니다
예를들어 10e+3 = 10 x 10^3 = 10 x 1000 = 10000
5e+2 = 5 x 10^2 = 5 x 100 = 500
10e-10 = 10 x 10^(-10) = 10 x 0.00000000...1 = 0.0000000...1
10e-3 = 10 x 10^(-3) = 10 x 0.001 = 0.010

그리고 float 은 마이너스 숫자가 없는게 아닙니다
16 진수 처럼 마이너스를 표현하기 때문에 그렇습니다

예제:

float per = 100.0;
float min = -3.0;
float good = 20000.00001;

bool 형
_________________________________________

true(참) 과 false(거짓) 만 있다는것을 나타내는 기호 입니다
논리 자료형 이라고도 부릅니다
다른값이 들어가면 오류가 납니다. 예를들어 실수
논리 자료형 사이즈는 1 비트 입니다 최대 0 에서 부터 1 까지 저장 가능합니다
정수형 값은 들어가도 상관 없습니다 그대신 0 에서 부터 1 안까지의 정수만 가능합니다
논리 자료형을 이용한 연산등이 많습니다

예제:

bool AreYouMan = true;
bool AreYouWoMan = false;
bool TF2IsMyLife = 1;
bool Nice = 0;
bool feel_good = 0;
bool feel_very_good = 1;

char 형
_________________________________________

문자 를 나타내는 기호 입니다
문자 사이즈는 1 바이트 입니다 최대 0 에서 부터 127 까지 저장 가능합니다
정수형 값은 들어가도 상관 없습니다 그대신 0 에서 부터 127 안까지의 정수만 가능합니다

예제:

char word = 'a';
char medic = 'E';
char good = 124;
char nope = 'n';
char null = 0;
