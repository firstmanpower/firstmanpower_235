while
_________________________________________

while(조건){

코드

for
_________________________________________

사용 방법:

for(변수; 조건; 계산)
{

코드

}

for 문은 조건이 false 가 될때까지 반복합니다

예제:



//1번째 반복: i 는 0과 같고 0 이 10보다 작으니깐 블락 안에 있는 코드를 실행시킵니다
//2번째 반복: i 는 1과 같고 1 이 10보다 작으니깐 블락 안에 있는 코드를 실행시킵니다
//3번째 반복: i 는 2과 같고 2 가 10보다 작으니깐 블락 안에 있는 코드를 실행시킵니다
//4번째 반복: i 는 3과 같고 3 이 10보다 작으니깐 블락 안에 있는 코드를 실행시킵니다
//5번째 반복: i 는 4과 같고 4 가 10보다 작으니깐 블락 안에 있는 코드를 실행시킵니다
//6번째 반복: i 는 5과 같고 5 가 10보다 작으니깐 블락 안에 있는 코드를 실행시킵니다
//7번째 반복: i 는 6과 같고 6 이 10보다 작으니깐 블락 안에 있는 코드를 실행시킵니다
//8번째 반복: i 는 7과 같고 7 이 10보다 작으니깐 블락 안에 있는 코드를 실행시킵니다
//9번째 반복: i 는 8과 같고 8 이 10보다 작으니깐 블락 안에 있는 코드를 실행시킵니다
//10번째 반복: i 는 9과 같고 9 가 10보다 작으니깐 블락 안에 있는 코드를 실행시킵니다
//i 는 10과 같고 10 이 10 보다 작은게 아니니깐 반복문 종료

int value;

for(int i; i < 10; i++) //의미: 변수 i 선언, 변수 i 가 10보다 작으면, i++ 를 수행 그리고 블락안에있는 코드 실행시킴
{

value++;

}
