Enum
_________________________________________

Enum 은 열거형 입니다

열거형을 사용하는 방법: 

enum 이름
{
	열거자 이름1,
	열거자 이름2,
	열거자 이름3,
	.
	.
	.
}

예제:

enum firstmanpower
{
    medic, //처음 시작은 0 에서 부터 시작합니다
    is, //1
    a, //2
    spy, //3
    LOL, //4
}

enum firstmanpowe2r
{
    medic = 1, //1 부터 시작 한다는 의미입니다
    is, //2
    a, //3
    spy = 2, //2 부터 시작 한다는 의미입니다
    LOL, //3
    MEM, //4
}
