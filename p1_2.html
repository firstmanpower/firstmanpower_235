배열

배열은 우리가 많이 선언해야할 변수를 단한번에 처리해주는 아주 고마운 친구입니다
_________________________________________

하는방법
_________________________________________

1.자료형 이름[양];
2.자료형 이름[양] = {값, ...};

2번째 방법은 배열을 초기화 하는 방법입니다
그리고 [양] 은 꼭 정수형 값이여야 합니다
실수 값은 오류가 납니다

2번째 하는 방법 예제:
int firstmanpower[100] = {9999, ...}; //이런식으로 하면 firstmanpower[0] 에서 부터 firstmanpower[99] 까지 다 9999 값으로 초기화 됩니다
//그리고 배열은 0 에서부터 시작합니다

-------------
int value1; |
int value2; |
int value3; |
int value4; | = int value[8];
int value5; |
int value6; |
int value7; |
int value8; |
-------------

보시다시피 얼마나 편리한지 아시겠죠?

그리고 배열은 0 에서 부터 시작합니다
그래서 만약 배열을 8 로 지정했다고 하면 0, 1, 2, 3, 4, 5, 6, 7 까지의 숫자만 되고 더이상의 숫자는 안됩니다


에러 예제:

int value[8];
value[-1] = 30; //value[-1] 은 불가능합니다, 배열은 0 에서 부터 시작합니다
value[0] = 301;
value[1] = 331;
value[2] = 0;
value[3] = 9999;
value[4] = 100000;
value[5] = -195;
value[6] = -44;
value[7] = 1983;
value[8] = 1; //value[8] 은 영역을 벗어납니다 만약 8 까지 하고 싶으시면 int value[9]; 로 다시 수정 해야합니다

2차원 배열 하는방법
_________________________________________

2차원 배열은 배열이 배열안에 있는것을 말합니다

1.자료형 이름[양][양];
2.자료형 이름[양][양] = {{값, ...}, ...};

2번째 방법은 2차원 배열을 초기화 하는 방법입니다

2번째 하는 방법 예제:
int firstmanpower[5][10] = {{8181, ...}, ...};

동적 배열
_________________________________________

일반 배열들은 꼭 상수로 지정하여야 오류가 안납니다
변수로 지정하실에는 오류가 납니다 예를들어
int ha = 1;
char test[30+ha]
그리고 ha 는 변수입니다. 그래서 배열에 ha 를 대입할시에 오류가납니다. 그대신에 ha 를 1 로 바꾸어서 char test[30+1] 로 하면 오류가 안납니다
왜냐하면 1 은 상수이기 때문입니다. 변수는 변할수있어서 안됩니다

이런 단점을 보안 하기 위해서 동적배열이 나타난겁니다

동적배열 하는 방법:

자료형[] 이름 = new 자료형[양];
자료형[][] 이름 = new 자료형[양][양];

예제:

int[] Medic = new int[30];
bool[] NOPE = new bool[4];
float[] height = new float[100];
char[] Words = new char[30];

예제 2:

int arraysize = 3+4+5+6+2;
int[] value = new int[arraysize]; //int value[3+4+5+6+2]; 와 같습니다
arraysize = arraysize + 21; 
float[] thanks = new float[arraysize]; //int value[3+4+5+6+2+21]; 와 같습니다

예제 3:

int[][] good = new int[5][5]; //int good[5][5]; 와 같습니다
float[][] Noice = new float[1][1]; //float Noice[1][1]; 와 같습니다
bool[][] hope = new bool[10][5]; //bool hope[10][5]; 와 같습니다
char[][] triple_words = new char[30][35]; //char triple_words[30][35]; 와 같습니다
